Privacy Policy – Reya Chrome Extension
Last updated: 3/6/2026

Reya Chrome Extension displays trading information from the Reya platform directly inside the browser.

Data Collection
This extension does not collect, store, or transmit personal data to the developer or any third parties.

Users may enter a wallet address and select a trading symbol in order to retrieve trading information from the official Reya API. This information is used only to display account data within the extension.

Local Storage
The extension uses Chrome’s storage API to store user preferences locally on the user’s device. This may include:

wallet address
selected trading symbol
cached position data
This data remains only on the user’s device and is never sent to the developer.

Third‑Party Services
The extension communicates only with official Reya services:

https://api.reya.xyz

https://ws.reya.xyz

These services are required to retrieve trading and market data.

Data Sharing
The extension does not sell, share, or transfer user data.

Changes
This policy may be updated in the future. Updates will be reflected on this page.
